Sudoku
Инструкция

Сетку судоку нужно записывать в отдельный текстовый докумен.
Каждый ряд с новой строки, цифры записывать через пробел, в конце ряда запятая не нужна.
Вместо пропусков нужно писать "0".

Программа решает судоку размера N*N, где N-это квадрат какого-либо числа.
Если сетка будет задана с иными параметрами, то программа выведет ошибку: "Sudoku is given incorrectly".

Метод creating_grid считывает данные из файла и создаёт сетку судоку.
Метод find_empty_position ищет пустые позиции, в котрые нужно будет вставить цифру.

Если судоку имеет не одно решение, то программа выводит все решения.
Если же судоку не имеет решений, то программа выводит: "No solutions"

Чтобы программа сама решила судоку, тогда нужно нажать кнопку "Solve".
Если же вы сами хотите заполнить сетку, то нужно кликать на пустые места в сетке,
вам будет показан список цифр, которые вы можете вставить.
Если цифра будет вставлена правильно, то клетка закрасится в зелёный цвет, иначе клетка 
закрашивается в красный цвет.
